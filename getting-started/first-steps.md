# First Steps

1. Open any existing material or create a new **Material Instance**.
2. Set Parent to `M_MaterialCraft` or any other `MI_LC_Instance` (e.g. `MI_LC_Foliage`).
3. Set the material **Blend Mode** to `Masked` if you are going to use Opacity.
4. Set the **Shading Model** to `Subsurface` if you are going to use Subsurface, or `TwoSidedFoliage` if you are going to use it as a Foliage Material.
5. Check **use LayerCraft** bool to `True` and assign your first LayerCraft layer from the provided library.
